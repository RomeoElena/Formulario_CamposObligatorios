# Ejercicio: Formulario con Validación Dinámica:

Desarrollo de una interfaz de usuario para capturar información, validando los datos mediante JavaScript, aplicando estilos con CSS y HTML.

## Requisitos del Formulario:

### Campos Solicitados:

- **Nombre completo:** Obligatorio.
- **Correo electrónico:** Obligatorio (con validación de formato).
- **Número de teléfono:** Opcional (solo debe aceptar números).
- **Fecha de nacimiento:** Obligatorio.
- **Género:** Selección mediante radio buttons.
- **País:** Menú desplegable con al menos 3 opciones.
- **Términos y condiciones:** Checkbox obligatorio.

## Funcionalidades y Lógica (JavaScript):

- **Validación de campos:** Al intentar enviar, los campos obligatorios vacíos se marcan con un **borde rojo**.
- **Confirmación:** Si todos los datos son correctos, se muestra el mensaje: _"Formulario enviado correctamente"_.
- **Gestión de eventos:** El envío se procesa capturando el evento de formulario para evitar la recarga de página innecesaria.

## Estilos y Diseño (CSS):

- **Interfaz Moderna:** Aplicación de márgenes, rellenos (padding) y bordes redondeados.
- **Interactividad:** Efecto visual (`hover`) en el botón de envío al pasar el cursor.
- **Responsividad:** El diseño debe ser atractivo y funcional en diferentes dispositivos.
